# Telco Customer Churn & AI Retention Assistant

Проект по прогнозированию оттока клиентов и генерации персонализированных предложений удержания с помощью LLM.

## Роли и стек
- **ML:** Python, Pandas, Scikit-Learn, LightGBM
- **AI:** LangChain / LLM API (для генерации текстовых инсайтов)
- **MLOps:** FastAPI, Pydantic v2, Docker, MLflow
